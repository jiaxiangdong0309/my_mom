# 知识库查询和管理

本技能用于检索、搜索和添加本地 SQLite 数据库中的记忆。

---

## 公共信息（必读）

### 重要提示
- **必须使用 `python3` 命令**
- **所有操作通过 API 进行**（默认端口：7937）
- **输出格式为 JSON**
- 首次使用或连接失败请运行：`mymom`
